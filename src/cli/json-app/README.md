# Json App CLI

Simple command-line app

- When user input message
- It will save the message to a file as JSON format

```bash

json-app -m "Hello world"

```

- and output will be

```json

{
  "message": "Hello world"
}

```

## Steps

- Command definition
- CLI Config
- Execute command

## Ingredients

- Arguments
- Options
- Valued Options
